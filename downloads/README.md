# 🛡️ HRIDOY TOP RAIDER v2.0

**The Ultimate Fast Discord Raiding Tool**

একটি শক্তিশালী এবং দ্রুতগতির ডিসকর্ড রেইডিং টুল যা টার্মাক্সের মাধ্যমে খুব সহজেই চালানো যায়। এতে রয়েছে মাস ব্যান, মাস কিক এবং অটোমেটিক স্প্যামিং সিস্টেম।

## 🚀 বৈশিষ্ট্য (Features)

* ✅ **Mass Ban:** এক ক্লিকে ১০০০ মেম্বার পর্যন্ত ব্যান।
* ✅ **Mass Kick:** দ্রুত সার্ভার খালি করার জন্য কিক সিস্টেম।
* ✅ **Mass Channel Delete:** সব চ্যানেল মুহূর্তেই ডিলিট।
* ✅ **Mass Create & Spam:** একসাথে অনেক চ্যানেল তৈরি এবং সেগুলোতে অটোমেটিক স্প্যাম মেসেজ পাঠানো।
* ✅ **Guild Info Change:** সার্ভারের নাম পরিবর্তন।
* ✅ **Turbo Speed:** মাল্টি-থ্রেডিং সিস্টেম যা কাজকে করে তোলে সুপার ফাস্ট।

## 🛠️ সেটআপ গাইড (Setup Guide)

নতুন ফোনে বা টার্মাক্সে এটি চালাতে নিচের কমান্ডগুলো ধাপে ধাপে দিন:

### ১. টার্মাক্স আপডেট ও পাইথন ইনস্টল:

```bash
pkg update && pkg upgrade -y
pkg install python -y

```

### ২. প্রয়োজনীয় লাইব্রেরি ইনস্টল:

```bash
pip install httpx colorama pystyle

```

### ৩. টুলটি চালানো:

```bash
python hridoy.py

```

## ⚠️ কিছু জরুরি সতর্কতা (Important Tips)

১. **Role Position:** বটের রোলটি সার্ভার সেটিংসে গিয়ে সবার উপরে তুলে দিন, নয়তো ব্যান/কিক কাজ করবে না।

২. **Intents:** [Discord Developer Portal](https://www.google.com/search?q=https://discord.com/developers/applications)-এ গিয়ে আপনার বটের **"Server Members Intent"** অবশ্যই **ON** করে দিন।

৩. **Permissions:** বটকে **Administrator** পারমিশন দিয়ে সার্ভারে অ্যাড করুন।
